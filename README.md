# ようこそ Qiitadon.Fans へ

## 主な目的

-  2022/02/28 の [Qiitadon.com の閉鎖](https://blog.qiita.com/terminate-provision-of-qiitadon/)に向けての各種アーカイブ先として。
-  異なるインスタンスに引っ越したとしても、キーたんの同窓として繋がれる先の 1 つとして。
